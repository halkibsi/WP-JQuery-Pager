==Short Description ==

Add a booklet with turnable pages.

==Description==

This plugin is a light template/framework for developers to easily add an booklet to a page/post using the WordPress gallery for that content. 

== Notes ==

This is a minimal plugin, function over form.  If you would like to extend it, or would like us to extend it in later versions, feel free to contact us at admins@ivycat.com.  We do custom plugins as well as upgrades and features for existing plugins.

== Installation ==

You can install from within WordPress using the Plugin/Add New feature, or if you wish to manually install:

1. Download the plugin.
1. Upload the entire `wp-jquery-paged` directory to your plugins folder 
1. Click to Activate the plugin in your WordPress plugin page

== Usage ==

1. Upload your images to the page or post.  Give them an order (IMPORTANT) 1-x
1. Use the following shortcode: `[wp-jquery-paged]`
1. That's it!
